# Finance Skill

## HOW: 환율 정보 처리 방법

이 스킬은 환율 정보를 조회하고 적용하는 방법을 정의합니다.

---

## 환율 조회 프로세스

### 1단계: 통화 코드 확인
사용자 요청에서 대상 통화 코드를 식별합니다. (예: "원화", "한국 돈" -> KRW)

### 2단계: 조회 도구 실행
`get_exchange_rate` 도구를 사용합니다.

**방법**:
```python
# USD -> KRW 조회
get_exchange_rate(target_currency="KRW", base_currency="USD")
```

### 3단계: 결과 적용
비용 계산이나 리포트에 환율 정보를 포함할 때 사용합니다.

**방법**:
```text
비용(USD) * 환율 = 비용(KRW)
```

---

## 검증 규칙

1. **유효한 통화 코드**: ISO 4217 표준 코드 사용 (USD, KRW, EUR, JPY 등)
2. **오류 처리**: 환율 조회 실패 시 원본 비용(USD 등)을 그대로 표시하고 조회 실패 메시지 포함
